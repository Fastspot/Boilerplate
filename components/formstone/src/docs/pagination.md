### Basic

Pagination will convert a set of links into a mobile-friendly pagination system:

```javascript
$(".pagination").pagination();
```

```markup
<nav class="pagination">
	<a href="1.html">1</a>
	<a href="2.html">2</a>
	<a href="3.html">3</a>
	<a href="4.html">4</a>
	<a href="5.html">5</a>
</nav>
```
